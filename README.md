# Исследование рынка общественного питания Москвы

**Описание проекта**

**Цель исследования** – провести комплексный исследовательский анализ рынка общественного питания Москвы для определения характеристик «типичного» заведения и поиска факторов, влияющих на его успех (рейтинг).

**Основные задачи**:
- Предобработка: Очистка данных, работа с пропусками и подготовка признаков для анализа.
- Портрет «типичного» заведения: Определение наиболее распространенных форматов и стандартов рынка.
- Анализ рейтингов: Изучение взаимосвязи столбца rating с другими характеристиками (тип заведения, количество мест, район).
- Поиск успешных практик: Выявление факторов, которые статистически значимо влияют на высокую оценку пользователей.
- Рекомендации: Формирование стратегии выбора типа и локации заведения на основе данных.

**Ключевые выводы**:
- Факторы рейтинга: Выявлены ключевые параметры (тип сервиса, масштаб), напрямую коррелирующие с пользовательским рейтингом.
- Типизация: Определены стандарты для различных форматов (среднее количество мест и ценовые ожидания).
- Локационная стратегия: Найдены закономерности распределения высокорейтинговых заведений по районам Москвы.

**Стек технологий:** Python (Pandas, NumPy, Phi_K, Missingno), визуализация (Seaborn, Matplotlib)

# Moscow Food Service Analysis

**Project Overview**

**Project Goal**: To conduct a comprehensive exploratory data analysis (EDA) of the Moscow food service market to identify "typical" establishment profiles and uncover factors influencing success (user ratings).

**Key Objectives:**
- Preprocessing: Data cleaning, handling missing values, and feature engineering.
- "Typical" Venue Profiling: Identifying the most common market formats and industry standards.
- Rating Analysis: Examining the relationship between the rating column and other variables (venue type, seating capacity, district).
- Best Practices Discovery: Identifying factors that statistically significantly impact high user ratings.
- Strategic Recommendations: Developing a data-driven strategy for choosing the type and location of a new venue.

**Key Insights:**
- Rating Drivers: Identified key parameters (service type, scale) that directly correlate with user satisfaction.
- Market Standardization: Established benchmarks for different formats (average seating capacity and price expectations).
- Location Strategy: Discovered distribution patterns of high-rated venues across Moscow districts.

**Tech Stack:** Python (Pandas, NumPy, Phi_K, Missingno), Visualization (Seaborn, Matplotlib)
